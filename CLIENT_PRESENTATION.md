## ğŸ” Intelligent Search System

### Hybrid Search Explained

Most AI systems use only one type of search. OpticWise uses **four complementary strategies**:

**1. Semantic Search (Vector)**
- Understands meaning and context
- Finds conceptually similar content
- Example: "infrastructure ownership" finds "network control" and "data ownership"

**2. Keyword Search (BM25)**
- Finds exact term matches
- Captures specific terminology
- Example: "PPP Audit" finds exact mentions

**3. Reciprocal Rank Fusion**
- Combines both search types
- No manual tuning needed
- Best of both worlds

**4. AI Re-Ranking**
- Claude Sonnet 4.5 reorders results
- Understands true relevance
- Human-like judgment

**Result:** 20-30% better search quality than vector-only systems
