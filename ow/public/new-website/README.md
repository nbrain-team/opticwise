# OpticWise New Website

## Overview
This is the new OpticWise website built with modern HTML/CSS and shadcn-inspired design principles.

## Design Approach
- **Aesthetic Inspiration**: Paycile.com - modern, clean, professional SaaS design
- **Color Scheme**: Professional blue (#0066CC) with green accent (#10B981)
- **Typography**: Inter font family for modern, clean readability
- **Layout**: Responsive, mobile-first design

## Technical Stack
- Pure HTML5
- CSS3 with Custom Properties (CSS Variables)
- No JavaScript dependencies (static site)
- Mobile-responsive design
- Cross-browser compatible

## Design Principles
1. **Spacing**: Consistent rhythm using 8px base unit
2. **Typography**: Clear hierarchy with Inter font
3. **Colors**: Limited palette for professional appearance
4. **Shadows**: Subtle elevation for depth
5. **Transitions**: Smooth 200ms animations
6. **Buttons**: Prominent CTAs with hover states
